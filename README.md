# Tsundere-Raws - Sous-Titres

Bonjour, bonsoir et bienvenue sur se repository qui regroupera les sous-titres que la Tsundere-Raws aura édité. Tout les sous-titres sont utilisable partout. Si vous remarquez qu'il y a une ou plusieurs fautes, n'hesitez pas à corriger et à nous envoyer une demande de merge ! Pensez à lire les `README.md` des sous-titres, des infos complementaires y sont proposé !

❗ Attention: Il n'y aura ici que des sous-titres édité par nous (avec parfois des ajout provenant d'autres sous-titres)

Pour toute demande autre, le [discord](https://discord.io/tsundere-raws) est ouvert !
